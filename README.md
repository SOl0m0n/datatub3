# Datatub3

Tool development to make PoC of Data Exfiltration HTTP/S (Client Version).

# Install

 $ git clone https://github.com/xf5/datatub3

 $ python3 datatub3.py

# Exfiltration Features
- User-Agent
- Cookie
- POST Request
- GET Request
- All requests encoded with Base 64

#Requirements
- Python 3
